# Playwright QA Report

- Date: 2026-03-20
- Environment: local Vite dev server `http://127.0.0.1:4173`
- Tooling: Playwright MCP
- Scope: 공개 라우트 접근, 비로그인 상태의 인증 필요 액션, `WeeklyInsightCard` 다크 테마 렌더링

## Summary

이번 스모크 QA에서 핵심 사용자 흐름은 정상 동작했다. 홈, 게시글 상세, 프로필 페이지는 비로그인 상태에서도 열렸고, 인증이 필요한 액션은 로그인 페이지로 이동하지 않고 안내 토스트만 노출됐다. `WeeklyInsightCard`는 다크 테마에서 토큰 기반 그라디언트가 적용된 상태로 렌더링됐다.
